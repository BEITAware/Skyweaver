# Title: UI Design does not conform to Aero aesthetics (Hardcoded Hex Colors and Flat Corners)

## Description
During a review of the UI design in the Skyweaver project, it was identified that several XAML files do not conform to the expected "Aero aesthetics". As per the design guidelines:
- Avoid hardcoded hex colors. Instead, use theme-defined dynamic resource bindings like `{DynamicResource AeroBackgroundBrush}`.
- Avoid flat corners (`CornerRadius="0"`). Instead, use theme-defined dynamic resource bindings like `{DynamicResource StandardCornerRadius}`.

The following files contain these issues and should be updated.
