您是否还在使用囿于命令行之中的传统人工智能代理应用程序？Ferrita开辟了一条不一样的路径。不同于业界泛滥的无头终端代理，Ferrita配备了精心设计的用户页面，旨在带给您来自未来的人工智能代理应用程序。Ferrita有着仿佛2020穿越而来的玻璃质感和美妙外观，它将人工智能带入了您的生活。您是否将运用人工智能变成了生活中不可少的享受？

## 配置语言模型从未像今天这么简单

Ferrita支持配置各类大语言模型API。通过直观的用户界面，您可以轻松地配置、管理和使用各种大语言模型。从OpenAI到Google，Ferrita支持大量语言模型的原生多模态能力并进行了深度集成。

## AerialCity：自研的向量图数据库与RAG基础设施

AerialCity是Ferrita的核心组件之一。它是一种轻量级向量数据库同时带有基础的图数据库支持。AerialCity支撑了本应用程序的语义搜索能力，使得代理能够轻松地搜索到与自己问题最相关的信息，即使这些信息的表述方式与用户的问题不同。不仅如此，AerialCity还被设计为一种混合检索数据库，支持向量检索与关键词检索的结合，代理可以根据自己的需求选择不同的检索方式。

## Persona与记忆：代理系统中的角色扮演能力

Persona是一套常用语和性格定义。它能够帮助代理在不同场景下保持角色的一致性。同时，Persona还能够与基于检索的记忆系统结合，有效保证跨越会话的记忆能力。
