# Issue: Aero Aesthetics Design Violations

## Description
According to the project's design guidelines for Aero aesthetics, UI elements should avoid using hardcoded hex colors and flat corners (`CornerRadius="0"`). Instead, they should utilize theme-defined dynamic resource bindings such as `{DynamicResource AeroBackgroundBrush}` and `{DynamicResource StandardCornerRadius}`.

This issue tracks the files and lines where these guidelines are currently violated.
